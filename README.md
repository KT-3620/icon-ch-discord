# `Icon` `Changer` `@Discord `

Discord 鯖のアイコンを変える Bot

## What’s this?

（現時点では）日替わりで Discord サーバーのアイコンを変える Bot です。いずれは設定を変えたりして

## Why this?

- スラッシュコマンド対応
- 最低限の権限での動作
  - 「サーバーを管理」権限のみで動く
  - メッセージの読み書きは一才しません
- カスタマイズ性抜群（にする予定）   

## Install

### 必要なもの

- Node.js
- pnpm
  - https://pnpm.io/ja/installation

### `1` 依存パッケージをインストール

```bash
pnpm install
# pnpm i
```
